# GNU Radio FM Transceiver

## Overview

This project implements a complete FM (Frequency Modulation) communication system using GNU Radio. The system captures real-time audio from a microphone, performs Wideband Frequency Modulation (WBFM), simulates a wireless communication channel using a Channel Model, demodulates the received signal, and reproduces the recovered audio through the speaker.

The project demonstrates the fundamental principles of Software Defined Radio (SDR), digital signal processing, and wireless communication systems.

---

## Features

- Wideband FM (WBFM) Transmission
- Wideband FM Reception
- Wireless Channel Simulation
- Real-Time Audio Recovery
- Time Domain Visualization
- Frequency Spectrum Visualization
- SDR-based Communication System
